Parágrafo

A tag <p> é uma das mais importantes do HTML. De forma bem simples, ela serve para criar um parágrafo de texto. Pense nela como a forma correta de escrever um parágrafo em um documento digital, assim como você faria em um livro ou em um editor de texto. Aqui estão os pontos principais para você entender como ela funciona:

  • Pra que serve?

    • A função principal da tag <p> é dar significado ao seu texto. Ela diz ao navegador: "o texto aqui dentro é um parágrafo".

  • É um Elemento de Blocos:

    • Isso significa que, por padrão, a tag <p> se comporta como uma caixa que ocupa a largura inteira da tela. É por isso que, sempre que você usa uma tag <p> , o texto seguinte automaticamente começa em uma nova linha, com um espaço extra acima e abaixo.

  • É Necessário Fechamento:

    • Sempre que abrir a tag com <p> e nescessário fechá-la com </p>. Todo o texto do seu parágrafo deve ficar entre elas.

  • Exemplo:

    • Imagine que você tem o seguinte código:

                
                
    Este é o primeiro parágrafo do meu site. Ele tem uma ideia completa e é formatado como um bloco de texto.
    
                <p>Este é o primeiro parágrafo do meu site. Ele tem uma ideia completa e é formatado como um bloco de texto.</p>
                
    Este é o segundo parágrafo. Ele começa automaticamente em uma nova linha, separado do primeiro.
    
                <p>Este é o segundo parágrafo. Ele começa automaticamente em uma nova linha, separado do primeiro.</p>

Em resumo, use a tag code class="tag"><p> sempre que precisar de um bloco de texto. Deixe o trabalho de "dar cor" ou "mudar o tamanho" para o CSS, mas a estrutura de parágrafo é responsabilidade do HTML

Quebra de Linha

Imagine que você está escrevendo um texto no computador. Quando você aperta a tecla Enter, o cursor pula para a linha de baixo. A tag de quebra de linha em HTML faz exatamente a mesma coisa.
