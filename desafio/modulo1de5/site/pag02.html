Inserindo Imagens

Inserindo Favicon

Um favicon (abreviação de "favorite icon", ou ícone de favoritos) é um pequeno ícone visual que representa o seu site.

Ele aparece em alguns lugares-chave no navegador como:

  • Na aba do navegador, ao lado do título da página.
  • Na lista de favoritos e no histórico.
  • Na tela inicial do celular, se o site for adicionado como um atalho.

Exemplo:

A função principal do favicon é ajudar o usuário a identificar rapidamente seu site entre várias abas abertas, reforçando a identidade visual e a marca.

Para adicionar um favicon, você usa a tag <link> no <head> do seu arquivo HTML, como no exemplo abaixo: 

            
                <head>
                <link rel="shortcut icon" href="favicon.ico" type="image/x-icon">
                <title>Inserindo Imagens</title>
                <head>

Dessa forma, o navegador sabe qual imagem deve usar como ícone para a sua página.

Inserindo Imagens de Forma Interna

Para inserir uma imagem em uma página HTML, você usa a tag <img>. Esta é uma tag "vazia", o que significa que ela não precisa de uma tag de fechamento (</img>). Em vez disso, todas as informações que ela precisa são passadas através de atributos.

Como Inserir Uma Imagem

Você precisa de dois atributos principais para que a imagem funcione corretamente:

  1. <src> (Source - Fonte): Este atributo é obrigatório. Ele indica o caminho (endereço) do arquivo da imagem que você quer mostrar.
  2. <alt>(Alternative Text - Texto Alternativo): Este atributo é essencial. Ele fornece uma descrição da imagem. Se a imagem não puder ser carregada por algum motivo, o texto do <alt> será exibido no lugar. Ele também é fundamental para a acessibilidade (para leitores de tela) e para o SEO.

Exemplo: 

            
                <imgsrc="caminho/para/sua/imagem.jpg" alt="Descrição da imagem para quem não puder vê-la">

Pontos Chave:

  • Caminho da Imagem (<src>): O valor pode ser um caminho relativo (ex: imagens/favicon.jpg) se a imagem estiver na sua própria pasta, ou um caminho absoluto (ex: https://www.site.com/imagem.jpg) se a imagem estiver em outro servidor.
  • Descrição (<alt>): Use sempre o alt. Ele é a "voz" da sua imagem para quem não consegue vê-la.
  • Tamanho: Embora você possa usar os atributos width e height no HTML, a forma mais moderna e recomendada de controlar o tamanho da imagem é usando CSS, pois isso facilita a criação de sites responsivos que se adaptam a diferentes telas.

Em resumo, a tag <img> é a porta de entrada para imagens no HTML, e os atributos src e alt são as chaves para que ela funcione de forma completa e acessível.

Inserindo Imagens de Forma Externa

O uso de imagens externas, ou seja, de URLs que não estão no seu próprio projeto, funciona de forma muito similar a quando você insere uma imagem interna. A grande diferença está em como o navegador encontra o arquivo.

Como Funciona a Inserção de Imagens Externas

  1. Caminho Absoluto: Em vez de dar um caminho relativo (por exemplo, imagens/foto.jpg), você fornece um caminho completo, que inclui o protocolo (http:// ou https://), o domínio (www.site.com.br) e a localização exata da imagem no servidor.
  2. Solicitação ao Servidor: Quando a página HTML é carregada, o navegador lê a tag <src> e o valor do atributo src. Em seguida, ele faz uma "solicitação" separada para o servidor onde a imagem está hospedada.
  3. Carregamento da Imagem: O servidor de origem envia o arquivo da imagem para o navegador do usuário, que então a exibe na posição correta da sua página.

Pontos de Atenção

  • Velocidade de Carregamento: A velocidade com que a imagem externa é carregada depende da conexão do seu usuário e da velocidade do servidor onde a imagem está. Se o servidor for lento, sua página pode demorar mais para carregar.
  • Disponibilidade: Se o site de origem remover ou mover a imagem, ela deixará de aparecer na sua página, exibindo apenas o texto do atributo <alt>.
  • Permissões: É importante verificar se você tem permissão para usar a imagem de outro site, para evitar problemas de direitos autorais.

            <img class="imagem-exemplo" src="https://upload.wikimedia.org/wikipedia/commons/thumb/c/c5/Html5_dise%C3%B1o_web.png/600px-Html5_dise%C3%B1o_web.png?20180726103553" alt="Logo HTML5" title="Logo HTML5">

Em resumo, a forma externa de inserir imagens é útil para referências rápidas ou se você não pode hospedar a imagem em seu próprio projeto. No entanto, o método interno (caminho relativo) é a melhor prática para imagens essenciais do seu site, pois lhe dá total controle sobre elas.
