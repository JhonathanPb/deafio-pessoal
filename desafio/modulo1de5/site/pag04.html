Principais Formatações de Textos

As tags de formatação de texto no HTML servem para dar significado e estilo a trechos de um texto. Cada tag tem um propósito específico, e usá-las corretamente torna seu código mais organizado e acessível.

Negrito / Destaque <strong>

  • Para que serve:

    • Indica que o conteúdo é de importância forte, urgência ou seriedade. O navegador, por padrão, exibe o texto em negrito. É a tag recomendada para dar destaque por ser semântica.

  • Exemplo:

    • A segurança dos dados é uma questão crucial para a empresa.

    <p>A segurança dos dados é uma questão <strong>crucial</strong> para a empresa.</p>

Itálico / Ênfase<em>

  • Pra que serve:

    • Coloca ênfase no texto, destacando-o de forma sutil. Pense nela como a sua voz alterando a entonação de uma palavra. Por padrão, o navegador exibe o texto em itálico.

  • Exemplo:

    • Eu realmente preciso que você salve o projeto agora.

    <p>Eu realmente <em> preciso </em> que você salve o projeto agora.</p>

Texto Marcado<mark>

  • Pra que serve:

    • Destaca um trecho de texto como se você tivesse usado um marca-texto. É ideal para realçar palavras-chave em um contexto específico, como o resultado de uma busca.

  • Exemplo:

    • Sua busca por "HTML" retornou a palavra HTML neste parágrafo.

    <p>Sua busca por "HTML" retornou a palavra <mark>HTML</mark> neste parágrafo.</p>

Para mudar a cor da tag <mark>, você precisa usar CSS. Por padrão, os navegadores aplicam um fundo amarelo e texto preto ao <mark>.Você pode alterar isso facilmente com as propriedades background-color (para a cor de fundo) e color (para a cor do texto).

Texto Grande e Pequeno<small>

  • Pra que serve:

    • Diminui o tamanho do texto para indicar que ele é "pequeno" por natureza, como letras miúdas, avisos legais, direitos autorais ou notas de rodapé. Não use para diminuir o texto visualmente; para isso, use CSS.

  • Exemplo:

    • Nossos termos de serviço. Atenção: Apenas para maiores de 18 anos.

    <p>Nossos termos de serviço. <small>Atenção: Apenas para maiores de 18 anos.</small></p>

Texto Deletado<del>

  • Pra que serve:

    • Marca um trecho de texto que foi removido de um documento. O navegador geralmente o exibe com uma linha sobre o texto (tachado).

  • Exemplo:

    • O preço original era R$ 50,00. Agora custa R$ 35,00.

    <p>O preço original era <del>R$ 50,00</del>. Agora custa R$ 35,00.</p>

Texto Inserido<ins>

  • Pra que serve:

    • Marca um trecho de texto que foi adicionado a um documento. É o oposto de <del>. Por padrão, o navegador o exibe sublinhado.

  • Exemplo:

    • Nosso novo plano agora inclui acesso a todas as plataformas.

    <p>Nosso novo plano agora inclui <ins>acesso a todas as plataformas</ins>.</p>

Texto Sobrescrito<sup>

  • Pra que serve:

    • Exibe o texto um pouco acima da linha, como em fórmulas matemáticas ou referências.

  • Exemplo:

    • A famosa equação de Einstein: E = mc2

    <p>A famosa equação de Einstein: E = mc<sup>2</sup></p>

Texto Subscrito<sub>

    Pra que serve:

Exibe o texto um pouco abaixo da linha, como em fórmulas químicas.

  • Exemplo:

    • A fórmula da água é H2O.

    <p>A fórmula da água é H<sub>2</sub>O.</p>

    Tags que Não São Comuns Usar (<b> e <i>)

    As tags <b> e <i> ainda existem no HTML, mas são desaconselhadas para a maioria dos casos. A principal razão é a semântica.

    • <b>(bold): Deixa o texto em negrito, mas não adiciona significado.
    • <i>(italic): Deixa o texto em itálico, mas não adiciona significado.

    A diferença para <strong> e <em> é sutil, mas fundamental:

    • <strong>diz: "Esse texto é importante." (O visual negrito é uma consequência.)
    • <em>diz: "Deixe esse texto com o visual de negrito."

    A melhor prática na web moderna é usar as tags que dão significado ao conteúdo (<strong>, <em>, <mark>) e deixar o CSS para controlar o visual, como o tipo de fonte, cor ou tamanho. Use <b> e <i> apenas em casos muito específicos, quando não há um significado extra a ser transmitido, como em nomes de espécies ou termos técnicos.
