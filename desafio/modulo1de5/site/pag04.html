Principais Formatações de Textos

As tags de formatação de texto no HTML servem para dar significado e estilo a trechos de um texto. Cada tag tem um propósito específico, e usá-las corretamente torna seu código mais organizado e acessível.

Negrito / Destaque <strong>

  • Para que serve:

    • Indica que o conteúdo é de importância forte, urgência ou seriedade. O navegador, por padrão, exibe o texto em negrito. É a tag recomendada para dar destaque por ser semântica.

  • Exemplo:

    • A segurança dos dados é uma questão crucial para a empresa.

    <p>A segurança dos dados é uma questão <strong>crucial</strong> para a empresa.</p>

Itálico / Ênfase<em>

  • Pra que serve:

    • Coloca ênfase no texto, destacando-o de forma sutil. Pense nela como a sua voz alterando a entonação de uma palavra. Por padrão, o navegador exibe o texto em itálico.

  • Exemplo:

    • Eu realmente preciso que você salve o projeto agora.

    <p>Eu realmente <em> preciso </em> que você salve o projeto agora.</p>

Texto Marcado<mark>

  • Pra que serve:

    • Destaca um trecho de texto como se você tivesse usado um marca-texto. É ideal para realçar palavras-chave em um contexto específico, como o resultado de uma busca.

  • Exemplo:

    • Sua busca por "HTML" retornou a palavra HTML neste parágrafo.

    <p>Sua busca por "HTML" retornou a palavra <mark>HTML</mark> neste parágrafo.</p>

Para mudar a cor da tag <mark>, você precisa usar CSS. Por padrão, os navegadores aplicam um fundo amarelo e texto preto ao <mark>.Você pode alterar isso facilmente com as propriedades background-color (para a cor de fundo) e color (para a cor do texto).
