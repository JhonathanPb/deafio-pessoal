Principais Formatações de Textos

As tags de formatação de texto no HTML servem para dar significado e estilo a trechos de um texto. Cada tag tem um propósito específico, e usá-las corretamente torna seu código mais organizado e acessível.

Negrito / Destaque <strong>

  • Para que serve:

    • Indica que o conteúdo é de importância forte, urgência ou seriedade. O navegador, por padrão, exibe o texto em negrito. É a tag recomendada para dar destaque por ser semântica.

  • Exemplo:

    • A segurança dos dados é uma questão crucial para a empresa.

    <p>A segurança dos dados é uma questão <strong>crucial</strong> para a empresa.</p>

Itálico / Ênfase<em>

  • Pra que serve:

    • Coloca ênfase no texto, destacando-o de forma sutil. Pense nela como a sua voz alterando a entonação de uma palavra. Por padrão, o navegador exibe o texto em itálico.

  • Exemplo:

    • Eu realmente preciso que você salve o projeto agora.

    <p>Eu realmente <em> preciso </em> que você salve o projeto agora.</p>

Texto Marcado<mark>

  • Pra que serve:

    • Destaca um trecho de texto como se você tivesse usado um marca-texto. É ideal para realçar palavras-chave em um contexto específico, como o resultado de uma busca.

  • Exemplo:

    • Sua busca por "HTML" retornou a palavra HTML neste parágrafo.

    <p>Sua busca por "HTML" retornou a palavra <mark>HTML</mark> neste parágrafo.</p>

Para mudar a cor da tag <mark>, você precisa usar CSS. Por padrão, os navegadores aplicam um fundo amarelo e texto preto ao <mark>.Você pode alterar isso facilmente com as propriedades background-color (para a cor de fundo) e color (para a cor do texto).

Texto Grande e Pequeno<small>

  • Pra que serve:

    • Diminui o tamanho do texto para indicar que ele é "pequeno" por natureza, como letras miúdas, avisos legais, direitos autorais ou notas de rodapé. Não use para diminuir o texto visualmente; para isso, use CSS.

  • Exemplo:

    • Nossos termos de serviço. Atenção: Apenas para maiores de 18 anos.

    <p>Nossos termos de serviço. <small>Atenção: Apenas para maiores de 18 anos.</small></p>

Texto Deletado<del>

  • Pra que serve:

    • Marca um trecho de texto que foi removido de um documento. O navegador geralmente o exibe com uma linha sobre o texto (tachado).

  • Exemplo:

    • O preço original era R$ 50,00. Agora custa R$ 35,00.

    <p>O preço original era <del>R$ 50,00</del>. Agora custa R$ 35,00.</p>

Texto Inserido<ins>

  • Pra que serve:

    • Marca um trecho de texto que foi adicionado a um documento. É o oposto de <del>. Por padrão, o navegador o exibe sublinhado.

  • Exemplo:

    • Nosso novo plano agora inclui acesso a todas as plataformas.

    <p>Nosso novo plano agora inclui <ins>acesso a todas as plataformas</ins>.</p>

Texto Sobrescrito<sup>

  • Pra que serve:

    • Exibe o texto um pouco acima da linha, como em fórmulas matemáticas ou referências.

  • Exemplo:

    • A famosa equação de Einstein: E = mc2

    <p>A famosa equação de Einstein: E = mc<sup>2</sup></p>

Texto Subscrito<sub>

    Pra que serve:

Exibe o texto um pouco abaixo da linha, como em fórmulas químicas.

  • Exemplo:

    • A fórmula da água é H2O.

    <p>A fórmula da água é H<sub>2</sub>O.</p>

    Tags que Não São Comuns Usar (<b> e <i>)

    As tags <b> e <i> ainda existem no HTML, mas são desaconselhadas para a maioria dos casos. A principal razão é a semântica.

    • <b>(bold): Deixa o texto em negrito, mas não adiciona significado.
    • <i>(italic): Deixa o texto em itálico, mas não adiciona significado.

    A diferença para <strong> e <em> é sutil, mas fundamental:

    • <strong>diz: "Esse texto é importante." (O visual negrito é uma consequência.)
    • <em>diz: "Deixe esse texto com o visual de negrito."

    A melhor prática na web moderna é usar as tags que dão significado ao conteúdo (<strong>, <em>, <mark>) e deixar o CSS para controlar o visual, como o tipo de fonte, cor ou tamanho. Use <b> e <i> apenas em casos muito específicos, quando não há um significado extra a ser transmitido, como em nomes de espécies ou termos técnicos.

    Outras Formatações de Texto no HTML

    Além das formatações básicas, o HTML oferece tags que dão um significado mais específico ao conteúdo. Elas são essenciais para organizar textos técnicos e citações.

    Código Fonte e Pré-Formatado

    Essas tags são perfeitas para exibir trechos de código em seus sites.

    • <code>: Usada para marcar um pequeno trecho de código dentro de um parágrafo. O navegador geralmente exibe o texto com uma fonte monoespaçada.
    • <pre>: Usada para um bloco de texto que já está pré-formatado. O navegador vai preservar todos os espaços, quebras de linha e tabulações exatamente como você os digitou. É ideal para blocos maiores de código.

      • Exemplo:

      Para criar um parágrafo, você usa o comando <p>.

                  
                namespace CalculadoraDeIdade
                {

                    class Program
                    {
                      
                        static void Main(string[] args)
                        {
                            
                            Console.WriteLine("Bem-vindo à calculadora de idade!");
                            Console.WriteLine("Por favor, digite o ano de seu nascimento:");

                           
                            string anoNascimentoTexto = Console.ReadLine();
                            int anoNascimento = int.Parse(anoNascimentoTexto);
                            int anoAtual = DateTime.Now.Year;
                            int idade = anoAtual - anoNascimento;

                            Console.WriteLine($"Você tem {idade} anos.");
                        }

    Citações

    • <blockquote>: Usada para citar um trecho longo de texto que foi retirado de outra fonte. O navegador costuma exibir a citação com um recuo (indentação) nas margens.
    • <q>: Usada para uma citação curta dentro de um parágrafo. O navegador geralmente adiciona as aspas automaticamente.

    Exemplo:

    Como disse Albert Einstein, a imaginação é mais importante que o conhecimento.

            
            <p>Como disse Albert Einstein, <q> imaginação é mais importante que o conhecimento.</q></p>

    Qualquer um que nunca cometeu um erro nunca tentou nada novo.

    — Albert Einstein
            
            <blockquote>
                <p>Qualquer um que nunca cometeu um erro nunca tentou nada novo.</p>
                <footer>— Albert Einstein</footer>
            </blockquote>

    Abrevoações

    • <abbr>: Usada para marcar uma abreviação ou um acrônimo. O atributo title é essencial aqui, pois ele exibe o significado completo da abreviação quando o usuário passa o mouse sobre ela.

      • Aprender HTML é o primeiro passo para criar sites.

              
            <p>Aprender <abbr title="HyperText Markup Language">HTML</abbr> é o primeiro passo para criar sites.</p>

    Texto Invertido

    • <bdo>: A tag bdo significa "Bi-Directional Override" (Substituição Bidirecional). Ela é usada para inverter a direção do texto, o que é útil em sites que misturam idiomas que se lêem da esquerda para a direita (como o português) com idiomas que se lêem da direita para a esquerda (como o árabe).

    Exemplo:

    O nome dele é José.

            
            <p>O nome dele é <bdo dir="rtl">José<bdo>.</p>

    Neste caso, a palavra "José" foi exibida da direita para a esquerda. O atributo dir com o valor "rtl" (right to left) faz essa inversão.

    Listas no HTML: Ordenadas e Não Ordenadas

    Lista Não Ordenadas <ul>

    No HTML, as listas são usadas para agrupar itens relacionados. Existem dois tipos principais, e a diferença entre eles está na importância da ordem.

    A lista não ordenada é usada quando a sequência dos itens não importa.

    • Tag: A lista inteira é envolvida pela tag <ul> (de "unordered list").
    • Itens: Cada item da lista é colocado dentro de uma tag <li> (de "list item").
    • Visual: O navegador exibe os itens com marcadores, geralmente bolinhas ou quadrados.

    Pense em uma lista de compras: a ordem dos itens não faz diferença para a sua finalidade.

    Exemplo:

    • Café
    • Pão
    • Leite
                
                <ul>
                    <li>Café</li>
                    <li>Pão</li>
                    <li>Leite</li>
                </ul>

    Lista Ordenada <ol>

    A lista ordenada é usada quando a sequência dos itens é importante

    1. Tag: A lista inteira é envolvida pela tag <ol> (de "ordered list").
    2. Itens: Assim como na lista não ordenada, cada item é colocado dentro de uma tag <li>
    3. Visual: O navegador exibe os itens com um contador, que pode ser números (o padrão), letras ou numerais romanos.

    Pense em uma receita de bolo, onde cada passo precisa ser seguido na ordem certa.

    Exemplo:

    1. Pré-aqueça o forno.
    2. Misture os ingredientes.
    3. Despeje a massa na forma.
    4. Asse por 40 minutos.
                
                <ol>
                    <li>Pré-aqueça o forno.</li>
                    <li></li>Misture os ingredientes.<li></li>
                    <li></li>Despeje a massa na forma.<li></li>
                    <li></li>Asse por 40 minutos.<li></li>
                </ol>

    Resumo

      Use
        para listas onde a ordem não é relevante (como uma lista de recursos, uma lista de tarefas, etc.). Use
          para listas onde a ordem é crucial (como rankings, instruções passo a passo, sequências, etc.).
